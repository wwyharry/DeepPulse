# DeepPulse TUI 使用指南

## 启动

```bash
# 方式一：通过 pip install 后的命令（推荐）
deeppulse-tui

# 方式二：直接运行模块
python -m agent.tui_cli
```

命令行参数：
```bash
deeppulse-tui --model MODEL    # 覆盖模型
deeppulse-tui --verbose        # 显示详细日志
deeppulse-tui --no-judge       # 禁用评判Agent
```

> **Windows 用户**：推荐使用 Windows Terminal，已自动处理中文编码。如遇乱码，请确保终端使用 UTF-8 编码（`chcp 65001`）。

## 界面布局

```
┌──────────────────────────────────────────────────────────────────────┐
│  ⚡ DeepPulse │ 模型: deepseek-reasoner │ 会话: abc123 │ R3 │ ⏱ 12s  │
├────────────┬──────────────────────────────────┬──────────────────────┤
│ 📊 自选股   │     💬 对话流                    │  🔧 工具面板         │
│ 💼 持仓     │                                  │  📊 数据摘要         │
│ 📈 市场概况 │                                  │                     │
└────────────┴──────────────────────────────────┴──────────────────────┘
```

### 左栏：自选股 / 持仓 / 市场概况

- **自选股**：显示股票名称、实时价格、涨跌幅，每 60 秒自动刷新
- **持仓**：显示持仓股票、股数、盈亏金额和比例，每 60 秒自动刷新
- **市场概况**：上证指数、深证成指、创业板指实时行情

### 中栏：对话流

- 用户消息（带时间戳）
- 主分析Agent（流式输出、thinking 折叠/展开、工具徽章）
- 评判Agent（综合评分、问题列表、数据核实）
- 会话统计（轮次/工具数/耗时）
- 确认按钮（评判后显示）

### 右栏：工具面板 / 数据摘要

- **工具面板**：正在执行的工具、已完成工具列表和耗时
- **数据摘要**：分析过程中获取的关键数据（RSI、MACD、价格等）

### 顶部状态栏

实时显示：模型名 | 会话ID | 推理轮次 | 耗时 | 工具数 | 状态

|--------|------|
| `Ctrl+Q` | 退出 |

| `Ctrl+L` | 清屏 |
| `Ctrl+H` | 显示帮助 |
| `Ctrl+J` | 手动触发评判Agent |

|------|------|
| `/help` | 显示帮助 |
| `/memory` | 查看记忆列表 |
| `/memory search <关键词>` | 搜索记忆 |

| `/memory knowledge` | 知识图谱 |
| `/strategy` | 列出战法 |
| `/strategy search <关键词>` | 搜索战法 |
| `/watchlist` | 查看自选股 |

| `/watchlist remove <代码>` | 移除自选股 |
| `/portfolio` | 查看持仓 |

## 评判Agent

### 触发方式

1. **手动触发**：输入 `/judge` 或按 `Ctrl+J`
2. **语言触发**：输入包含"评判"、"帮我看看"、"检查一下"等关键词

### 评测维度

1. **数据核实** — 核对引用数据与工具返回是否一致
2. **推理链完整性** — 是否充分利用数据、观点是否自洽
3. **工具使用合理性** — 是否调用了足够工具、是否遗漏
4. **风险覆盖** — 是否提及风险、是否过度绝对化
5. **操作建议合理性** — 建议是否有数据支撑

### 输出格式

```
📊 评测报告

━━━ 综合评分: 7.5/10 ━━━

✅ 做得好的：...
⚠️ 需要注意的 N 个问题：...
📊 数据核实：✅ X条准确 / ⚠️ Y条存疑 / ❌ Z条错误
🎯 改进建议：...
```

### 评判后操作

- **✅ 认可**：自动记录到学习记忆
- **❌ 不认可**：忽略此次评判

## 文本复制

按住 **Shift 键** + 鼠标选择文本，然后 `Ctrl+Shift+C` 复制。

推荐使用 Windows Terminal 或 Windows Terminal Preview。
