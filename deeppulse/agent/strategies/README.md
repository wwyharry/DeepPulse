# 短线战法大全

在此目录下放置短线战法的 Markdown 文件，Agent 会自动加载并在分析时参考。

## 文件命名规范

建议使用以下命名格式：
```
01-放量突破战法.md
02-缩量回踩战法.md
03-均线金叉战法.md
...
```

## 文件内容格式建议

每个战法文件应包含以下结构（供 Agent 精准匹配）：

```markdown
# 战法名称

## 适用场景
描述该战法适用的市场环境和股票类型

## 核心条件
- 条件1：...
- 条件2：...

## 买入信号
具体的买入触发条件

## 卖出信号
止盈止损条件

## 关键指标
涉及的技术指标及其参数（如 MA5/MA10 金叉、RSI<20 等）

## 典型案例
历史案例说明（可选）

## 风险提示
该战法的风险点和注意事项
```

## 注意事项

- 文件编码请使用 UTF-8
- 每个文件聚焦一个战法，不要混合多个战法
- 关键条件和指标数值要明确，便于 Agent 精准匹配
- 可随时新增文件，重启 CLI 后自动生效
